# Travel Memories プライバシーポリシー

最終更新日: 2024年1月

## はじめに

Travel Memories（以下、「本アプリ」）は、ユーザーのプライバシーを最優先に考えています。このプライバシーポリシーは、本アプリがどのような情報を収集し、どのように使用するかを説明します。

## 収集する情報

### 1. 位置情報
- **収集する情報**: GPSによる現在地情報
- **使用目的**: 思い出の記録時に現在地を取得し、記録に紐付けるため
- **保存場所**: ユーザーのデバイス内のみ
- **第三者提供**: なし

### 2. 写真・動画
- **収集する情報**: ユーザーが選択した写真・動画ファイル
- **使用目的**: 思い出の記録として保存するため
- **保存場所**: ユーザーのデバイス内のみ
- **第三者提供**: なし

### 3. ユーザー入力情報
- **収集する情報**: タイトル、説明、タグ、評価など
- **使用目的**: 思い出の詳細情報として記録するため
- **保存場所**: ユーザーのデバイス内のみ
- **第三者提供**: なし

## 情報の使用

本アプリで収集された全ての情報は、以下の目的でのみ使用されます：

1. 思い出の記録・管理機能の提供
2. アプリ内での検索・並び替え機能の提供
3. 地図上での位置情報表示

## 情報の保存と管理

- 全てのデータはユーザーのデバイス内にのみ保存されます
- 外部サーバーへのデータ送信は行いません
- ユーザーはいつでもアプリ内でデータを削除できます
- アプリをアンインストールすると、全てのデータが削除されます

## 第三者サービスの利用

本アプリは以下の第三者サービスを利用しています：

### 地図サービス
- **サービス名**: Apple Maps (iOS) / Google Maps (Android)
- **利用目的**: 地図の表示、位置情報の検索
- **収集情報**: 地図表示時の位置情報のみ
- **プライバシーポリシー**: 各プラットフォームのポリシーに準拠

## 子供のプライバシー

本アプリは全年齢を対象としており、13歳未満の子供からも個人情報を意図的に収集することはありません。

## セキュリティ

私たちは、ユーザーの情報を保護するために商業的に合理的な予防措置を講じています。ただし、インターネットを介した送信方法や電子保存方法は100％安全ではないことをご理解ください。

## 権限の説明

### 位置情報アクセス
- **理由**: 思い出に現在地を記録するため
- **タイミング**: ユーザーが「現在地」ボタンをタップした時のみ

### 写真ライブラリアクセス
- **理由**: 思い出に写真・動画を追加するため
- **タイミング**: ユーザーがメディアを選択する時のみ

### カメラアクセス
- **理由**: 新しい写真・動画を撮影するため
- **タイミング**: ユーザーがカメラ機能を使用する時のみ

## ユーザーの権利

ユーザーは以下の権利を有します：

1. **アクセス権**: 保存された全ての情報を確認できます
2. **修正権**: いつでも情報を編集・更新できます
3. **削除権**: いつでも情報を削除できます
4. **ポータビリティ**: データはデバイス内に保存されているため、自由に管理できます

## プライバシーポリシーの変更

本プライバシーポリシーは随時更新される可能性があります。重要な変更がある場合は、アプリ内でお知らせします。

# Travel Memories Privacy Policy

Last Updated: January 2024

## Introduction

Travel Memories (the "App") prioritizes user privacy. This Privacy Policy explains what information the App collects and how it is used.

## Information We Collect

### 1. Location Information
- **What we collect**: GPS location data
- **Purpose**: To record current location when creating memories
- **Storage**: Only on user's device
- **Third-party sharing**: None

### 2. Photos and Videos
- **What we collect**: Photos and videos selected by users
- **Purpose**: To save as part of memories
- **Storage**: Only on user's device
- **Third-party sharing**: None

### 3. User Input
- **What we collect**: Titles, descriptions, tags, ratings
- **Purpose**: To record memory details
- **Storage**: Only on user's device
- **Third-party sharing**: None

## Use of Information

All information collected is used solely for:

1. Providing memory recording and management features
2. Enabling search and sort functionality
3. Displaying locations on maps

## Data Storage and Management

- All data is stored only on the user's device
- No data is sent to external servers
- Users can delete data anytime within the app
- Uninstalling the app removes all data

## Third-Party Services

The App uses these third-party services:

### Map Services
- **Service**: Apple Maps (iOS) / Google Maps (Android)
- **Purpose**: Map display and location search
- **Data collected**: Location data only when displaying maps
- **Privacy**: Subject to platform policies

## Children's Privacy

The App is suitable for all ages and does not intentionally collect personal information from children under 13.

## Security

We take commercially reasonable precautions to protect user information. However, no internet transmission or electronic storage method is 100% secure.

## Permission Explanations

### Location Access
- **Why**: To record current location with memories
- **When**: Only when user taps "Current Location"

### Photo Library Access
- **Why**: To add photos/videos to memories
- **When**: Only when user selects media

### Camera Access
- **Why**: To capture new photos/videos
- **When**: Only when user uses camera feature

## User Rights

Users have the right to:

1. **Access**: View all stored information
2. **Rectification**: Edit and update information
3. **Erasure**: Delete information anytime
4. **Portability**: Data is stored locally for free management

## Policy Changes

This Privacy Policy may be updated periodically. Significant changes will be notified within the app.
